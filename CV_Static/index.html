Proyectos

API BCRA | 1º Proyecto que compara la evolución del dólar Oficial vs Dólar Blue

1º Proyecto que usando un API del BCRA, Python permite hacer seguimiento de la evolución durante los últimos 4 años de inflación y dólar. Involucra evolución de la brecha entre dólar Oficial y dólar Blue. A nivel técnico se usa Python, Dataframe, Regresión Lineal, Seaborn.

COVID-19 | EE.UU 2º Proyecto individual.

El CDC (centro de control y prevención de enfermedades) de EE. UU. es la entidad encargada de monitorear la salud pública y desarrollar estrategias para la prevención y control de enfermedades. Por esto ha contratado a nuestra consultora para organizar, en base a los datos recolectados, los recursos hospitalarios para prevenir que lo ocurrido durante la pandemia COVID-19 suceda de vuelta. A nivel técnico se usa STREAMLIT, Python, POWER BI, Regresion Lineal.

IMPACTO CO2 | Proyecto Grupal

Proyecto Grupal, Se pretende analizar y estimar el impacto de la Huella de Carbono, junto con el consumo de combustibles y como se establece la correlación entre consumo y generación de contaminantes. A nivel de datos se usó el método de trabajo SCRUM, Diagrama de Entidad Relación, Obtención de Insights, Se implementó un servidor local Apache con My-SQL, para pruebas, el deploy se realizó en S3, Athena. La actualización de los datos se usó Crawlers (AWS), en un Bucket, Función Lambda. La visualización se usó POWER BI, embebido en un proyecto de STREAM-LIT. Se usaron varios modelos de Machine Learning (KNN, Desicion Trees), para clasificar datos. Se usó un modelo de red neuronal para forecasting. Proyecto Grupal, deploy StreamLit
